### Task 3 — Per-target import-sourcemap tests (commit 4fd8136)

Per-target import-sourcemap accuracy on component-graphs without `<script>` is a v2 follow-up (VALIDATION.md "hard-to-prove" residual).

**V1 contract per test:**

1. The synthesized import line resolves via `SourceMapConsumer.originalPositionFor` to a Modal.rozie line that is NOT line 1.
2. The resolved source line falls within the `<rozie name="Modal">...</rozie>` envelope.
3. The resolved `source` is non-null and matches `/Modal\.rozie$/`.

**v2 follow-up:** the strict tightening to "must fall WITHIN `<components>` block byte range" requires the script-block overwrite to be split into framing + body chunks; documented inline in commit message + the `shell.ts` forward-compat anchor (empty-output overwrite of the `<components>` block source range).

- **Commit:** 4fd8136 (Task 3).
